Our Heritage

The K2 Story

From a singular vision born at the foot of Canada's most majestic range, K2 Hotel Canada was conceived not merely as a place to rest, but as a monument to the extraordinary. Our founders believed that the mountains themselves could teach us what true luxury means: silent power, enduring grace, and the breathtaking beauty of the unconquered.

Every stone in our walls, every thread in our linens, and every flame in our kitchens carries forward this founding philosophy — that luxury is not about excess, but about the precision of a singular, perfect experience.

Today, K2 Hotel Canada stands as a testament to that vision — a five-star sanctuary where the ancient spirit of the mountains meets the finest contemporary craftsmanship, and where every guest discovers their own personal summit.

insert into k2_ceo (id) values ('main') on conflict do nothing; -- Rooms/Suites table create table if not exists k2_rooms ( id uuid primary key default gen_random_uuid(), name text not null, tier text default 'Signature Collection', price text default 'From CAD $1,000 / night', description text, amenities text default '', image_url text, available boolean default true, sort_order int default 0, created_at timestamptz default now() ); -- Facilities/Experiences table create table if not exists k2_facilities ( id uuid primary key default gen_random_uuid(), name text not null, description text, icon_type text default 'default', image_url text, learn_text text default 'Explore', sort_order int default 0, active boolean default true, created_at timestamptz default now() ); -- Gallery table create table if not exists k2_gallery ( id uuid primary key default gen_random_uuid(), image_url text not null, caption text default '', sort_order int default 0, created_at timestamptz default now() ); -- Hotel story table create table if not exists k2_story ( id text primary key default 'main', heading text default 'The K2 Story', para1 text, para2 text, para3 text, image_url text, updated_at timestamptz default now() ); insert into k2_story (id) values ('main') on conflict do nothing; -- Enable Row Level Security alter table k2_settings enable row level security; alter table k2_ceo enable row level security; alter table k2_rooms enable row level security; alter table k2_facilities enable row level security; alter table k2_gallery enable row level security; alter table k2_story enable row level security; -- Public read policies (frontend can read) create policy "Public read settings" on k2_settings for select using (true); create policy "Public read ceo" on k2_ceo for select using (true); create policy "Public read rooms" on k2_rooms for select using (true); create policy "Public read facilities" on k2_facilities for select using (true); create policy "Public read gallery" on k2_gallery for select using (true); create policy "Public read story" on k2_story for select using (true); -- Authenticated write policies (admin can edit) create policy "Auth write settings" on k2_settings for all using (auth.role() = 'authenticated'); create policy "Auth write ceo" on k2_ceo for all using (auth.role() = 'authenticated'); create policy "Auth write rooms" on k2_rooms for all using (auth.role() = 'authenticated'); create policy "Auth write facilities" on k2_facilities for all using (auth.role() = 'authenticated'); create policy "Auth write gallery" on k2_gallery for all using (auth.role() = 'authenticated'); create policy "Auth write story" on k2_story for all using (auth.role() = 'authenticated'); -- Storage buckets (run after creating buckets in Supabase dashboard) -- Bucket names: k2-logos, k2-ceo, k2-rooms, k2-gallery, k2-facilities, k2-story -- All buckets: Public = true -- Then add this storage policy for each bucket: -- create policy "Public storage read" on storage.objects for select using (bucket_id in ('k2-logos','k2-ceo','k2-rooms','k2-gallery','k2-facilities','k2-story')); -- create policy "Auth storage write" on storage.objects for insert using (auth.role() = 'authenticated' and bucket_id in ('k2-logos','k2-ceo','k2-rooms','k2-gallery','k2-facilities','k2-story')); -- create policy "Auth storage delete" on storage.objects for delete using (auth.role() = 'authenticated');

2. Go to Storage → Create 6 buckets: k2-logos, k2-ceo, k2-rooms, k2-gallery, k2-facilities, k2-story. Set each to Public.
3. Go to Authentication → Add your admin email/password user.
